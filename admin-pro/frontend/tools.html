关于 REITs 数据平台

REITs 数据平台是一个开放式金融数据平台，无需登录即可使用全部功能。

平台特色

  • • 实时行情数据 - 跟踪所有 REITs 基金
  • • 板块热力图 - 直观展示涨跌分布
  • • 智能对比 - 多维度基金对比分析
  • • 公告追踪 - 重要信息及时掌握

数据来源：新浪财经、东方财富、AKShare
