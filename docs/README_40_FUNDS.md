## 使用方法

### 一键重置（推荐）

```bash
# Windows
d:\tools\rixian\reset_40_funds.bat

# 或命令行
cd d:\tools\rixian
reset_40_funds.bat
```

### 手动执行

```bash
cd d:\tools\消费看板5（前端）\backend
node reset_40_funds.js
```

### 使用SQL文件

```bash
# 在SQLite命令行中
sqlite3 backend/database/reits.db
.read backend/database/reits_40_funds_complete.sql
```

## 文件说明

| 文件 | 位置 | 说明 |
|------|------|------|
| `reset_40_funds.bat` | `rixian/` | Windows一键重置脚本 |
| `reset_40_funds.js` | `rixian/` + `backend/` | Node.js重置脚本 |
| `reits_40_funds_complete.sql` | `rixian/` + `backend/database/` | 完整SQL脚本 |
| `README_40_FUNDS.md` | `rixian/` | 使用说明 |

## 注意事项

⚠️ **警告**: 重置会清空所有历史数据，请谨慎操作！

重置后需要重新获取行情数据：
```bash
cd d:\tools\消费看板5（前端）\backend
npm run crawl:price
```
